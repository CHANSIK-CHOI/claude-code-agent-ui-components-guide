States 쇼케이스에서 상태명 라벨은 반드시 버튼 **위에** 표기해야 한다. 버튼 아래 표기 금지.

**Why:** 에이전트 스펙에 "상태명 라벨 상단 표기"가 명시되어 있으며, 사용자가 버튼 아래 표기를 틀린 결과물로 지적했다.

**How to apply:** States 쇼케이스 생성 시 항상 라벨 y = 버튼 y - 22, 버튼이 라벨 아래에 위치하도록 배치.

---

쇼케이스 섹션명은 반드시 단순하게 통일한다:
- "States" (❌ "States · primary / md" 같은 부제 금지)
- "Modes" (❌ "Modes · icon & label" 같은 부제 금지)
- "Variant × Size"

**Why:** 사용자가 수식어 붙인 섹션명을 원하지 않는다고 수정 요청함.

**How to apply:** 모든 컴포넌트 쇼케이스 소제목에서 "·" 이후 부연 설명 추가 금지. 지정된 3개 이름만 사용.
